# Drone-H743 飞控架构

> 面向 STM32H743 + FreeRTOS 的共轴双桨矢量推力飞控。本文档基于 drawio 架构蓝图，定义软件分层、模块职责与数据流边界。

---

## 1. 总体架构

飞控软件分为 **6 层**，自上而下为：

```
┌─────────────────────────────────────────┐
│                应用层 (APP)              │
├─────────────────────────────────────────┤
│              控制器层 (Controller)       │
├─────────────────────────────────────────┤
│              中间件层 (Middleware)       │
├─────────────────────────────────────────┤
│               驱动层 (Driver)            │
├─────────────────────────────────────────┤
│              BSP / HAL 层               │
├─────────────────────────────────────────┤
│              MCU (STM32H743)             │
└─────────────────────────────────────────┘
```

前三层（APP、Controller、Middleware）合称 **Module**，是平台无关的纯软件逻辑。后两层（Driver、BSP/HAL）与硬件直接相关。

---

## 2. 硬件与驱动层

驱动层封装所有外设芯片的寄存器操作与通信协议，向上提供设备无关的数据接口。

### 2.1 传感器

| 模块 | 芯片 | 接口 | 输出数据 |
|---|---|---|---|
| IMU | ICM-42688 | SPI | 陀螺仪 (rad/s)、加速度计 (m/s²)、温度 |
| 磁力计 | - | I2C/SPI | 地磁矢量 (μT) |
| 气压计 | SPL06 | I2C/SPI | 气压 (Pa)、温度 |
| GPS | - | UART | 经纬度、高度、速度、星数 |

### 2.2 存储

| 模块 | 芯片/接口 | 用途 |
|---|---|---|
| FLASH | GD25Q32 (SPI) | 日志持久化存储 |
| SD Card | SDIO | 参数文件、配置文件 |

### 2.3 执行器与 IO

| 模块 | 接口 | 说明 |
|---|---|---|
| 电机 | PWM / ESC | 共轴双桨主旋翼 ×2 |
| 舵机 | PWM | 矢量推力舵机 |
| RC 遥控 | UART / PPM | 遥控器信号输入 |
| WIFI | UART | 地面站无线通信 |
| LED | GPIO | 状态指示 |

### 2.4 BSP / HAL

BSP 将驱动层与具体板卡绑定，HAL 由 CubeMX 生成。

| 子层 | 内容 |
|---|---|
| BSP | SPI/TIM/GPIO 引脚绑定、DMA 通道分配、CS/DRDY 中断映射、PWM 定时器通道映射、输出门控 |
| HAL | SPI / UART / I2C 外设句柄、TIM PWM / GPIO EXTI、DMA / MPU / Cache 配置 |

## 3. 控制器软件层

控制器层是飞控的核心实时链路，数据流如下：

```
传感器预处理 ──→ EKF ──→ 位置环P ──→ 速度环PID ──→ 姿态环P ──→ 角速度环PID ──→ 混控器 ──→ 电机/舵机
                    │                                                         │
                    └──→ 状态/安全 ←────────────────────────────────────────────┘
```

### 3.1 传感器预处理

- 负责从驱动层读取各传感器原始数据
- 执行零偏校准、量纲转换、坐标系旋转
- 对 IMU 数据进行时间戳标记
- 输出对齐后的观测数据给 EKF

### 3.2 EKF（扩展卡尔曼滤波）

- 融合 IMU、磁力计、气压计、GPS 多源数据
- 输出延迟但准确的姿态四元数、速度、位置、陀螺零偏估计
- 提供状态协方差 / 质量标志

### 3.3 级联控制器

四级串级 PID 结构，外环输出作为内环目标：

| 环 | 类型 | 输入 | 输出 |
|---|---|---|---|
| 位置环 | P | 位置误差 | 速度目标 |
| 速度环 | PID | 速度误差 | 姿态目标 |
| 姿态环 | P | 姿态误差 | 角速度目标 |
| 角速度环 | PID | 角速度误差 | 抽象控制力矩 |

控制器只输出抽象控制量（推力/力矩），不关心具体执行器几何。

### 3.4 混控器

- 将抽象控制力矩分配给具体执行器（电机油门、舵机角度）
- 共轴双桨矢量分配逻辑（非线性/饱和/舵机几何）
- 输出执行器 setpoint（PWM 占空比或舵机角度）

### 3.5 状态/安全模块

- 接收传感器预处理和 RC 遥控信号
- 对混控器输出进行安全门控
- 负责解锁/上锁判断、姿态超限检测、通信超时保护
- 输出门控信号控制电机/舵机使能

---

## 4. 中间件层

中间件层提供飞控运行所需的公共服务，不涉及控制算法。

### 4.1 LOG 日志记录

- 连接 FLASH 驱动，实现黑匣子记录
- 记录关键飞行数据：状态估计、控制输出、传感器原始值、事件
- 支持事后回读与分析

### 4.2 QCS 地面站（MAVLINK）

- 连接 WIFI 驱动，实现无线遥测
- MAVLINK 协议封装：姿态、位置、状态下行，命令上行
- 支持 QGroundControl 等标准地面站软件

### 4.3 参数管理器

- 连接 SD Card 驱动，实现参数持久化
- 存储 PID 增益、校准参数、限幅值等
- 支持参数在线修改与周期边界生效

---

## 5. 应用层

应用层负责飞控的任务调度、模式管理和高层决策。

- **飞行模式管理**：自稳、定高、定点、返航等模式切换
- **任务调度**：协调控制器层和中间件层的运行时序
- **状态机**：地面待命 → 解锁 → 起飞 → 飞行 → 降落 → 上锁
- **安全策略**：电池低电压保护、GPS 丢失保护、遥控信号丢失保护

## 7. 文件结构

```
App/
  Inc/
    app_controller.h        # 控制器层入口
    app_ekf.h               # EKF 状态估计
    app_mixer.h             # 混控器
    app_state_safety.h      # 状态/安全模块
  Src/
    app_controller.c
    app_ekf.c
    app_mixer.c
    app_state_safety.c

Middleware/
  Inc/
    mw_logger.h             # LOG 日志记录
    mw_mavlink.h            # MAVLINK 地面站
    mw_param.h              # 参数管理器
  Src/
    mw_logger.c
    mw_mavlink.c
    mw_param.c

Driver/
  Inc/
    drv_imu.h               # ICM-42688
    drv_mag.h               # 磁力计
    drv_baro.h              # SPL06
    drv_gps.h               # GPS
    drv_flash.h             # GD25Q32
    drv_sdcard.h            # SD Card
    drv_rc.h                # RC 遥控
    drv_motor.h             # 电机 ESC
    drv_servo.h             # 舵机
    drv_wifi.h              # WIFI
    drv_led.h               # LED
  Src/
    drv_imu.c
    drv_mag.c
    drv_baro.c
    drv_gps.c
    drv_flash.c
    drv_sdcard.c
    drv_rc.c
    drv_motor.c
    drv_servo.c
    drv_wifi.c
    drv_led.c

BSP/
  Inc/
    bsp_board.h             # 板级引脚/外设绑定
  Src/
    bsp_board.c

Core/                        # CubeMX 生成，不手改
  Src/
    freertos.c
    stm32h7xx_it.c
    main.c
```
